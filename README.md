# Personal Portfolio 

En personlig portfolio-webbplats byggd med HTML, CSS och JavaScript för att visa upp mina projekt, erfarenheter och färdigheter.

## Om projektet

Detta är min personliga portfolio-webbplats där jag presenterar:
- Om mig själv
- Projekt jag har byggt
- Teknologier jag använder
- Kontaktinformation

Syftet med projektet är att ge en **visuell presentation** av min kompetens som utvecklare och ge besökare ett enkelt sätt att utforska vad jag skapat.

## Projektstruktur

Här är de viktigaste filerna och mapparna i repot:

├── index.html # Startsida / Presentation
├── about.html # Om mig-sektionen
├── project1.html # Exempelprojekt 1
├── project2.html # Exempelprojekt 2
├── project3.html # Exempelprojekt 3
├── project4.html # Exempelprojekt 4
├── style.css # Allmän styling
├── main.js # JavaScript-logik
└── img/ # Bilder och tillgångar
