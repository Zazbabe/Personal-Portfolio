KORT BESKRIVNING AV UPPGIFTEN

"Create a simple web app for our customers that displays a list of orders (or parcels) with some details
they need to know like order status, ETAs, pickup locations, etc. Feel free to have UI/UX that best fits the use-case.

API Endpoint: https://my.api.mockaroo.com/orders.json?key=e49e6840

The app MUST be done using ReactJS and support major mobile & desktop browsers/layouts."

Min struktur/planering:
Detta projektet var det största av de 4 och hade fokus på att lära sig React.
Jag valde det då det gav mig ny kunskap i hur jag kan implementera ett API och använda mig av det i applikationen.

Tekniker som jag utforskat:
React, CSS, Grid, API, dynamiska CSS-klasser mm
